# Machine Learning Study

이 저장소는 「핸즈온 머신러닝」과  
김성범 교수의 「핵심 머신러닝」 강의를 기반으로  
머신러닝의 핵심 개념과 구현 과정을 정리한 학습 기록입니다.

이론을 단순히 요약하는 데 그치지 않고,  
직접 코드로 구현하고 실험하며 이해한 내용을 중심으로 정리합니다.

---

## 📌 학습 목표
- 머신러닝의 기본 개념과 수학적 직관 이해
- 주요 알고리즘의 동작 원리 파악
- Python 기반 머신러닝 구현 능력 향상
- 이후 딥러닝·컴퓨터 비전 학습을 위한 탄탄한 기초 구축
